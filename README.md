# #一个短链接应用

这是一个短链接应用，应用结构比较简单，用户管理，用户注册（当前分为两种形式，邮箱获取验证码，管理员邀请码），用户密码重置，短链接生成，短链接管理，不同角色拥有不同的短链接条目数，短链接访问统计，该项目需配合short_url_frontend项目使用（开发中）

🚩需要第三方应用:mysql,redis
